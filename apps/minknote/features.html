Features

Everything MinkNote can do, thoughtfully built for macOS power users.

Local-First File Ownership

MinkNote stores notes as standard text files in normal folders, not in a proprietary cloud container. You can inspect, back up, version, and move your files with familiar macOS tools.

Learn more

Multiple Projects

Each project is just a folder, but MinkNote treats it like a dedicated workspace. Keep work, personal, and research notes separate and switch between them without mixing contexts.

Learn more

Journals and Folder Hierarchy

MinkNote uses a straightforward model: journals at the top, folders inside journals, and notes inside those folders. Simple enough to scan quickly, structured enough to scale with your knowledge base.

Two Note Styles

Use standard notes for durable, continuously edited content, and journal-style notes for chronological capture. Timestamp-based filenames keep journal entries unique and ordered.

Learn more

Dual-Mode Editor

MinkNote lets you edit the same note in WYSIWYG or Markdown mode and move between them instantly. One source of truth, the editing style that fits the moment.

Formatting Toolbar
