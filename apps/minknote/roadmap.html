Roadmap

Planned feature set for MinkNote.

Core Editing & Authoring

11 / 20 done
  • Add notes to Favourites
  • Dual-mode editing with WYSIWYG and Markdown in one editor
  • Switch between WYSIWYG and Markdown with ⌘E
  • GitHub-Flavored Markdown support, including task lists, tables, and strikethrough
  • YAML Frontmatter management for note metadata
  • Live Markdown preview
  • Markdown syntax highlighting
  • Optional auto-linking for plain URLs
  • Inline code and code block rendering
  • Comfortable writing overscroll for long documents
  • Auto-save and manual save preferences
  • Pinning notes to top
  • Right-hand panel with document sections (outline)
  • Ability to move tasks up/down with arrow keys
  • Code syntax highlighting
  • Links between notes
  • External file changes cause updates in MinkNote
  • Ability to collapse header sections of text
  • Right-hand panel with "stats": word count, reading time, file created/modified
  • Audio dictation
  • Handwriting iPad Pencil integration

UI / UX & Personalisation

13 / 25 done
  • Ability to hide formatting toolbar in Prefs
  • Customise Journal icons: set colour, icon, sync selection colour
  • Drag images from Finder directly into notes
  • Paste images from the clipboard into notes
  • Store images in organised sibling i/ folders
  • Move notes with their associated images
  • Resolve duplicate image filenames automatically
  • Three-level journal, folder, and sub-folder hierarchy
  • Right-click context menus for folders, journals, and notes
  • Drag notes between journals and folders in the sidebar
  • Dark Mode with system appearance detection and manual override
  • Switch between one, two, and three-pane layouts
  • Ability to double-click on a note to open it up in a new window.
  • Add items to Filter section in sidebar (Inbox, Trash, etc.)
  • Custom ordering of journals
  • Highlight keywords in search results
  • Customisable App icon
  • Filter note list by tags, tasks, images, etc
  • Ability to show images/documents/files in folders
  • Previews for embedded content like YouTube links
  • Support for multiple themes, fonts, font sizing
  • Drag images from one note to another in note list
  • Drag PDFs into notes
  • Drag files onto note list or a particular journal to create notes
  • Support Mermaid diagrams

Syncing, Platforms & Privacy

5 / 7 done
  • Full multi-vault support
  • Direct filesystem storage for compatibility with external tools
  • Standard Markdown file formats: .md, .txt, and .markdown
  • Sync by placing projects in iCloud Drive or another cloud folder
  • Works with Dropbox, Google Drive, OneDrive, and other cloud storage services
  • Companion iOS and iPadOS apps
  • Private notes with biometric support
